# ROBOTS.md

## Проект

Журнал лучника — мобильное PWA-приложение в стиле iOS для учета тренировок по стрельбе из лука.

## Правила разработки

- Основной язык интерфейса: русский.
- Приоритет: iPhone/Safari/PWA.
- UI должен быть близок к нативному iOS: серый фон, белые карточки, системная типографика, аккуратные отступы.
- Не ломать уже стабильные сценарии ввода серий.
- Все изменения делать небольшими понятными коммитами.
- Перед изменениями в логике ввода проверять сценарии:
  - последовательное заполнение серий;
  - редактирование заполненной стрелы;
  - очистка только последней заполненной серии;
  - комментарии к сериям;
  - импорт/экспорт истории и экипировки.
